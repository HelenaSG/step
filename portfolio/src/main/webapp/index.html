Voronoi Diagram Image Converter

This project is a Mathematica implementation that changes images to Voronoi diagrams. Generally, the code takes in a local image as input, converts it into a binary image, generates random points from the binary image, changes the points to Voronoi diagrams, then colors the diagram as the final output. On the first row are the input images, and on the second row are the Voronoi diagrams generated from 500 randomly selected sites (shown as orange dots). The third row shows the artistically colored images. The last two rows are results of 5000 or 10000 sites.
